## devkitとは？

Maya devkitを使うことで、Mayaのプラグインを開発したり、スタンドアロン アプリケーションを作成して Maya の機能を拡張できます。  
その他にも、Mayaスクリプトのインテリセンスを利かせたり、スクリプトサンプルから学習したりと幅広く利用用途があります。
